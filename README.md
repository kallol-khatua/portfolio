# Kallol Khatua Portfolio
* <a href="https://kallolkhatua.vercel.app/">Go to portfolio</a>

## Run on local system
* Clone the repo
   ```sh
   git clone https://github.com/kallol-khatua/portfolio.git
   ```
* Install NPM packages
   ```sh
   npm install
   ```
* Run
    ```sh
   npm run dev
   ```

## Run on localhost
Go to the browser and paste the link
```sh
http://localhost:5173/
```

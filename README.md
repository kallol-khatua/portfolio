## Run on local system

- Clone the repo
  ```sh
  git clone https://github.com/kallol-khatua/portfolio.git
  ```
- Install NPM packages
  ```sh
  npm install
  ```
- Run
  ```sh
  npm run dev
  ```
- Go to the browser and paste the link
  ```sh
  http://localhost:5173/
  ```
